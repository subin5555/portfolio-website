Book Store

The Book Store website is a responsive front-end project built using only HTML and CSS. It presents a simple yet attractive layout where users can browse books by categories such as fiction, non-fiction, and bestsellers. The homepage highlights featured books with cover images, descriptions, and pricing, while navigation links make it easy to explore different sections. Using pure CSS, the design includes custom layouts, typography, colors, and hover effects without relying on frameworks. The website also includes an About Us page, Contact section, and space for customer testimonials, creating a complete and user-friendly browsing experience. This project demonstrates strong knowledge of core web technologies and the ability to design functional websites without external libraries.

Gusto

The Gusto Coffee website is a modern and responsive front-end project created using HTML and Tailwind CSS. It showcases a variety of coffee items such as espresso, cappuccino, and lattes with stylish layouts and smooth hover effects. Tailwind’s utility-first classes were used to build a clean design, responsive grid system, and reusable components for the menu, about section, and contact page. The homepage highlights featured coffee items with attractive images, pricing, and call-to-action buttons, making it user-friendly and visually appealing. This project demonstrates the ability to build professional-looking websites with Tailwind CSS, focusing on simplicity, responsiveness, and a great user experience.

BMW Clone

The BMW Clone website is a responsive front-end project built using HTML, CSS, and Bootstrap. It replicates the modern design of BMW’s official website with sections for featured cars, specifications, image galleries, and promotional banners. Using Bootstrap’s grid system and components, the website ensures a clean, mobile-first layout with smooth navigation across devices. Custom CSS styling was applied for branding consistency, hover effects, and animations, making the site visually engaging. This project highlights skills in building real-world website clones, responsive design, and integrating Bootstrap effectively for professional UI development.
